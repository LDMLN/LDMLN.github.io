Drawing App

This app was made using C++ and Open Frameworks. I wanted to practice some graphics programming and I love drawing so this seemed a good use of my time.

You can draw your own image and take screenshots; altering the background and brush type and color, like any image drawing app.

However, there are also some more "wacky" features like making each point on your brush bounce around the screen animated. You can also randomize the colors and alter if only outlines get drawn. There are tons of small features that users can interact and tinker with.

Additionally, there is a very simple image generation feature included in this app. If a user selects the "random drawing" option at the main menu, a series of random coordinates are generated and plotted. Lines are then drawn with alternating brush types, colors, and sizes between each following point.
